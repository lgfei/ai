## 项目概述

已成功为"有声"（YouSheng）品牌创建了一个专为老年人设计的安卓文字转语音应用。应用采用 Kotlin 开发，基于 Android TextToSpeech API，具有简洁直观的界面和老年人友好的设计。

## 主要特性

### 老年人友好设计
1. **大字体**：标题 28sp，正文 20sp，按钮 24sp+
2. **大按钮**：最小 64dp 高，200dp 宽
3. **高对比度**：深色文字配浅色背景
4. **竖屏锁定**：避免误操作
5. **简化的操作流程**：只有 4 个主要按钮

### 核心功能
1. **文字转语音**：支持中文语音合成
2. **开始朗读**：一键朗读输入的文字
3. **暂停/继续**：可在朗读过程中暂停和继续
4. **停止朗读**：立即停止当前朗读
5. **清空文本**：快速清除输入框内容

### 技术实现
- 使用 Android TextToSpeech API
- 语速设置为 0.8（稍慢，适合老年人）
- 优先使用中文语音（Locale.CHINA）
- 支持 API 21+（Android 5.0+）

## 下一步操作

### 在 Android Studio 中打开
1. 打开 Android Studio
2. File → Open → 选择当前目录 (yousheng/)
3. 等待 Gradle 同步完成
4. 连接安卓设备或启动模拟器
5. 点击运行按钮

### 首次运行前
- 确保设备已安装中文语音包（通常系统自带）
- 如果没有中文语音，可在设置中下载或使用应用商店安装

- 验证界面在不同屏幕尺寸上的显示
