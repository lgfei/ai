【核心功能】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 文字转语音 (TTS)
   ✓ 支持中文语音合成
   ✓ 使用 Android TextToSpeech API
   ✓ 语速优化为 0.8（适合老年人）

2. 用户界面
   ✓ 大字体设计（标题 28sp，正文 20sp，按钮 24sp）
   ✓ 大按钮设计（最小 64dp 高）
   ✓ 高对比度色彩方案
   ✓ 竖屏锁定

3. 操作功能
   ✓ 开始朗读 - 绿色按钮
   ✓ 停止朗读 - 红色按钮
   ✓ 暂停/继续 - 橙色按钮
   ✓ 清空文本 - 辅助按钮

4. 用户体验
   ✓ 状态实时显示
   ✓ 按钮状态智能控制
   ✓ 欢迎消息提示

【使用方法】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 输入文字 - 在文本框中输入或粘贴要朗读的文字
2. 点击【开始朗读】- 绿色按钮，开始语音朗读
3. 点击【暂停】- 橙色按钮，暂停朗读
4. 点击【继续】- 再次点击橙色按钮继续
5. 点击【停止】- 红色按钮，停止朗读
6. 点击【清空】- 清除文本框内容


【注意事项】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
