# 优声应用 - 老年人友好版

## 项目概述

这是一个专为老年人设计的安卓文字转语音应用，具有以下特点：

- 大字体、大按钮设计
- 高对比度色彩方案
- 简单直观的操作流程
- 支持中文语音朗读

### 核心功能实现

#### 1. 文字转语音 (TTS)

使用 Android 内置的 `TextToSpeech` API：

```kotlin
private lateinit var textToSpeech: TextToSpeech

override fun onInit(status: Int) {
    if (status == TextToSpeech.SUCCESS) {
        textToSpeech.setLanguage(Locale.CHINA)
        textToSpeech.setSpeechRate(0.8f)  // 语速稍慢，适合老年人
    }
}
```

#### 2. 界面设计要点

- 字体大小：标题 28sp，正文 20sp，按钮 24sp
- 按钮尺寸：最小 64dp 高，200dp 宽
- 触摸目标：确保至少 48dp
- 对比度：深色文字配浅色背景

#### 3. 老年人友好特性

- 竖屏锁定，避免误操作旋转
- 简化按钮：只有读、停、暂停/继续、清空四个操作
- 状态提示：实时显示语音引擎状态
- 自动提示：首次打开显示欢迎消息

### 构建和运行

1. **在 Android Studio 中打开项目**
2. **同步 Gradle**（File → Sync Project with Gradle Files）
3. **连接设备或启动模拟器**
4. **点击运行按钮**（绿色三角形）

或者使用命令行：

```bash
./gradlew assembleDebug          # 构建调试版本
./gradlew installDebug           # 安装到设备
adb shell am start -n com.yousheng.tts/.MainActivity  # 启动应用
```

### 测试要点
