📋 项目概述
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
这是一个专为老年人设计的安卓文字转语音应用，具有：
✓ 大字体、大按钮设计，方便阅读和点击
✓ 高对比度界面，减少视觉疲劳  
✓ 简单直观的操作：一键朗读、暂停、停止
✓ 支持中文语音合成
✓ 竖屏锁定，适合单手操作


🎯 如何运行应用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法一：使用 Android Studio（推荐）

1. 下载并安装 Android Studio
   官网：https://developer.android.com/studio

2. 打开 Android Studio

3. 选择 "Open an Existing Project"

4. 选择项目目录：
   /mnt/e/Test/llm/yousheng

5. 等待 Gradle 同步完成（首次打开需要下载依赖）

6. 连接安卓手机或启动模拟器

7. 点击绿色运行按钮（▶）


方法二：命令行构建

# 构建调试版 APK
./gradlew assembleDebug

# 安装到连接的设备
./gradlew installDebug

# 或使用 adb
adb install app/build/outputs/apk/debug/app-debug.apk


📱 首次运行前的准备
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 确保设备已安装中文语音包
   - 大多数安卓手机系统自带中文语音
   - 如果没有，可在 "设置 → 语言和输入法 → 文字转语音"
     中下载中文语音包

2. 允许未知来源应用安装（如果手动安装 APK）
   - 设置 → 安全 → 未知来源 → 允许


🎮 使用方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 在文本框中输入或粘贴要朗读的文字

2. 点击【开始朗读】按钮（绿色）- 开始朗读

3. 点击【暂停】按钮（橙色）- 暂停朗读
   再次点击可继续朗读

4. 点击【停止】按钮（红色）- 停止朗读

5. 点击【清空】按钮 - 清除文本框内容

💡 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 语音无法朗读中文？
A: 请在手机设置中安装中文语音包：
   设置 → 语言和输入法 → 文字转语音 → 安装中文语音

Q: 按钮太小/文字太小？
A: 本应用已针对老年人优化，如仍觉得小，
   可在手机设置中调整系统字体大小。

Q: 如何调整朗读速度？
A: 目前版本固定为适中的语速（0.8倍速，适合老年人）。
   后续版本会添加语速调节功能。

Q: 应用闪退？
A: 请确保：
   1. Android 版本 >= 5.0
   2. 已安装语音引擎
   3. 给应用授予必要的权限
